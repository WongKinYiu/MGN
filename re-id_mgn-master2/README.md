## Remark

(1)在caffe2darknet的转换中，slice层由于DarkNet不支持，需要修改DarkNet源码添加新层实现

(2)需要修改源码让max_pool层支持多尺度kernel size和stride size

(3)修改DarkNet源码，增加了Python接口调用对numpy格式输入的支持
